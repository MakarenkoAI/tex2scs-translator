# Tex2SCs-translator

Tex2SCs-translator is a tool for translating SCn-represented *.tex files into SCs-represented *.scs files.

## Quick start

## Usage

  ```sh
  #build translator
  ./scripts/make_all.sh
  #translate files
  ./bin/scn-tex2scs -s <path to folder with scn-tex sources> -t <target folder for translated scs sources>
  ```
